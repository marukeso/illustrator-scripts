# illustrator-scripts

Adobe Illustrator を操作する ExtendScript（`.jsx`）置き場。
選択オブジェクトのレイヤー整理など、繰り返し作業をショートカット一発で行うためのスクリプトをまとめる。

## スクリプト一覧

| ファイル | 内容 |
| --- | --- |
| `move-img.jsx` | 選択オブジェクトを `img` レイヤーへ移動（無ければ作成） |
| `move-asset.jsx` | 選択オブジェクトを `asset` レイヤーへ移動（無ければ作成） |
| `_shared.jsxinc` | 上記の共通処理。単体では使わない |

move 系は複数選択に対応。選択したオブジェクトをまとめて移動できる。

### 移動（move）の挙動
- 選択オブジェクトを小文字レイヤー（`img` / `asset`）へ移動する。
- 該当レイヤーが無ければ **自動作成**してから移動する。
- レイヤーがロック／非表示のときは中断して知らせる（解除してから再実行）。
- 実行後に **移動した件数**を通知するので、選んだ枚数と照合して取りこぼしに気づける。

### 廃止：書き出し（export）スクリプト
`export-img.jsx` / `export-asset.jsx`（レイヤー内オブジェクトの PNG・JPEG 一括書き出し）は**廃止した**。
共通処理の `exportLayer()` は `_shared.jsxinc` に残してあるので、必要になれば `git restore` でエントリ2本を復活できる。

> `.jsx` と `_shared.jsxinc` は**同じフォルダに置いたまま**にすること（`#include` で相対参照している）。
> このリポジトリのフォルダごと好きな場所に置けばよい（下記いずれの実行方法でも相対参照は解決される）。

## 実行方法

### A. すぐ試す（Presets 不要・管理者権限不要）
任意の場所（このリポジトリのまま）から都度実行できる。
- **ファイル > スクリプト > その他のスクリプト…**（`Ctrl+F12`）で `.jsx` を選んで実行。
- または `.jsx` を Illustrator のウィンドウへ**ドラッグ&ドロップ**。

この方法は「都度実行」専用。**アクションには記録できない**（下記の落とし穴を参照）。

### B. アクション＋再生ボタン（▶）で常用する ※要インストール
アクションに記録するには、スクリプトを **Presets のスクリプトフォルダにインストール**しておく必要がある（管理者権限が必要）。

1. `.jsx` と `_shared.jsxinc` を、Illustrator のスクリプトフォルダへコピーする（例）:

   ```
   C:\Program Files\Adobe\Adobe Illustrator [バージョン]\Presets\ja_JP\スクリプト\
   ```

   `Program Files` 配下は管理者権限が要る。権限が無ければ IT 担当に上記コピーを依頼する。
   `_shared.jsxinc` もメニューには出ないが `#include` で必要なので一緒に入れる。
2. Illustrator を再起動 → 「ファイル > スクリプト」メニューに並ぶ。
3. 「ウィンドウ > アクション」でアクションパネルを開き、新規アクション作成 → 記録開始。
4. **ファイル > スクリプト > 対象スクリプト**（メニューに並んだもの）を実行。
5. ■（記録停止）。各スクリプトごとに別アクションを作る（`move-img` / `move-asset`）。

以降は、アクションパネルで対象アクションを選んで再生ボタン（▶）を押せば実行できる。

> **落とし穴：アクション記録は「その他のスクリプト…」を記録してくれない。**
> アクションに記録できるのは Scripts メニューに名前で並ぶ＝インストール済みのスクリプトだけ。
> 「単体では動くのにアクションに入らない」場合はこれが原因。B の手順でインストールすること。
>
> インストール後にスクリプトを更新したら、スクリプトフォルダへ再コピー（上書き）が必要。

## 開発メモ

- 文字コード・改行は LF。
- スクリプトの動作確認は Illustrator 上で実際に選択 → 実行して行う。
- 大きめの変更の前には必ず1コミット（`git restore` で戻せる安全網）。
