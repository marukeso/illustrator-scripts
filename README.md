# illustrator-scripts

Adobe Illustrator を操作する ExtendScript（`.jsx`）置き場。
選択オブジェクトの命名など、繰り返し作業をショートカット一発で行うためのスクリプトをまとめる。

## スクリプト一覧

| ファイル | 内容 |
| --- | --- |
| `rename-to-img.jsx` | 選択中の全オブジェクトの名前を `img` にする |
| `rename-to-asset.jsx` | 選択中の全オブジェクトの名前を `asset` にする |
| `export-assets-png.jsx` | （おまけ）選択オブジェクトを PNG でフォルダに書き出す |

いずれも複数選択に対応。選択したオブジェクトを1つずつ処理するので、まとめて選んで一括実行できる。

## ショートカット一発で起動する設定

Illustrator はスクリプトへ直接ショートカットを割り当てられないため、**アクション経由**でファンクションキーに割り当てる。

### 1. スクリプトを登録
`.jsx` を Illustrator のプリセットフォルダに置く（例）:

```
C:\Program Files\Adobe\Adobe Illustrator [バージョン]\Presets\ja_JP\スクリプト\
```

Illustrator を再起動すると「ファイル > スクリプト」メニューに表示される。
（コピーしたくなければ、アクション記録時に「参照」からこのフォルダのファイルを直接指定してもよい）

### 2. アクションに記録してキー割り当て
1. 「ウィンドウ > アクション」でアクションパネルを開く
2. 新規アクション作成 → ファンクションキー（例: F2）を割り当て → 記録開始
3. 「ファイル > スクリプト > 対象スクリプト」を実行
4. ■（記録停止）
5. 別スクリプトは別アクションを作り、別のキー（F3 など）を割り当てる

## 開発メモ

- 文字コード・改行は LF。
- スクリプトの動作確認は Illustrator 上で実際に選択 → 実行して行う。
