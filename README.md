# illustrator-scripts

Adobe Illustrator を操作する ExtendScript（`.jsx`）置き場。
選択オブジェクトの命名＋画像書き出しなど、繰り返し作業をショートカット一発で行うためのスクリプトをまとめる。

## スクリプト一覧

| ファイル | 内容 |
| --- | --- |
| `export-img.jsx` | 選択オブジェクトを `img` にリネーム → `img` レイヤーへ移動 → PNG と JPEG で書き出す |
| `export-asset.jsx` | 選択オブジェクトを `asset` にリネーム → `asset` レイヤーへ移動 → PNG と JPEG で書き出す |
| `_shared.jsxinc` | 上記2本の共通処理（リネーム＋移動＋書き出し）。単体では使わない |

いずれも複数選択に対応。選択したオブジェクトを1つずつ処理するので、まとめて選んで一括実行できる。

### 移動の挙動
- オブジェクト名と同じ小文字レイヤー（`img` / `asset`）へ移動する。
- 該当レイヤーが無ければ **自動作成**してから移動する。
- レイヤーがロック／非表示だと移動に失敗する（その旨をまとめて通知。リネームと書き出しは実行される）。

### 書き出しの挙動
- 書き出し先は **開いているドキュメントと同じ場所の `assets/` サブフォルダ**（無ければ自動作成）。
- 1オブジェクトにつき `名前-01.png` と `名前-01.jpg` を書き出す（例: `img-01.png` / `img-01.jpg`）。
- 連番は `-01` から始まる2桁ゼロ埋め（`img-01`, `img-02` …）。既に同じ番号があれば空いている次の番号へ進む（上書きしない）。
- 連番の順番は **アートボードの上→下**（同じ高さのものは左→右）。行の判定幅は `_shared.jsxinc` の `_ROW_TOL`（初期値10pt）で調整できる。
- オブジェクトの範囲（アートボード）でクリップするため、範囲内に重なる別オブジェクトも写り込む点に注意。
- ドキュメントが未保存のときだけ、書き出し先を選ぶダイアログが出る。

> `_shared.jsxinc` は `.jsx` と**同じフォルダ**に置くこと（`#include` で相対参照している）。
> Presets フォルダへコピーして使う場合は、`.jsx` と `_shared.jsxinc` をまとめてコピーする。

## アクション再生で起動する設定

Illustrator はスクリプトへ直接ショートカットを割り当てられないため、**アクションに記録**しておき、アクションパネルの再生ボタン（▶）で呼び出す。

### 1. スクリプトを登録
`.jsx` を Illustrator のプリセットフォルダに置く（例）:

```
C:\Program Files\Adobe\Adobe Illustrator [バージョン]\Presets\ja_JP\スクリプト\
```

Illustrator を再起動すると「ファイル > スクリプト」メニューに表示される。
（コピーしたくなければ、アクション記録時に「参照」からこのフォルダのファイルを直接指定してもよい）

### 2. アクションに記録
1. 「ウィンドウ > アクション」でアクションパネルを開く
2. 新規アクション作成 → 記録開始
3. 「ファイル > スクリプト > 対象スクリプト」を実行
4. ■（記録停止）
5. 別スクリプトは別アクションを作る

以降は、アクションパネルで対象アクションを選んで再生ボタン（▶）を押せば実行できる。
