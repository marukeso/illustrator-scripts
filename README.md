# illustrator-scripts

Adobe Illustrator を操作する ExtendScript（`.jsx`）置き場。
選択オブジェクトのレイヤー整理や画像書き出しなど、繰り返し作業をショートカット一発で行うためのスクリプトをまとめる。

## スクリプト一覧

「レイヤーへ移動」と「書き出し」を分けてある。**まず move で移動 → レイヤーパネルで全部移ったか確認 → export で書き出す**、という流れで使う。

| ファイル | 内容 |
| --- | --- |
| `move-img.jsx` | 選択オブジェクトを `img` レイヤーへ移動（無ければ作成） |
| `move-asset.jsx` | 選択オブジェクトを `asset` レイヤーへ移動（無ければ作成） |
| `export-img.jsx` | `img` レイヤー内の全オブジェクトを PNG と JPEG で書き出す（選択不要） |
| `export-asset.jsx` | `asset` レイヤー内の全オブジェクトを PNG と JPEG で書き出す（選択不要） |
| `_shared.jsxinc` | 上記4本の共通処理。単体では使わない |

move 系は複数選択に対応。選択したオブジェクトをまとめて移動できる。

### 移動（move）の挙動
- 選択オブジェクトを小文字レイヤー（`img` / `asset`）へ移動する。
- 該当レイヤーが無ければ **自動作成**してから移動する。
- レイヤーがロック／非表示のときは中断して知らせる（解除してから再実行）。
- 実行後に **移動した件数**を通知するので、選んだ枚数と照合して取りこぼしに気づける。

### 書き出し（export）の挙動
- 対象は選択ではなく **`img` / `asset` レイヤー内のオブジェクト全部**（グループは1つのアセットとして扱う）。
- 書き出し先は **開いているドキュメントと同じ場所**の、レイヤー名と同じサブフォルダ（`export-img` → `img/`、`export-asset` → `asset/`。無ければ自動作成）。
- 1オブジェクトにつき **等倍と2倍**を書き出す。ファイル名は同じで、フォルダで分ける。例（`export-img`）:
  - `img/img-01.png` / `img/img-01.jpg`（等倍）
  - `img/2x/img-01.png` / `img/2x/img-01.jpg`（2倍・Retina 用）
- 連番は `-01` から始まる2桁ゼロ埋め（`img-01`, `img-02` …）。等倍フォルダと2倍フォルダの両方の空きを見るので、ペアは常に揃う。既に同じ番号があれば空いている次の番号へ進む（上書きしない）。
- 連番の順番は **アートボードの上→下**（同じ高さのものは左→右）。行の判定幅は `_shared.jsxinc` の `_ROW_TOL`（初期値10pt）で調整できる。
- 各オブジェクトを一時ドキュメントに複製して、その可視範囲（バウンディングボックス）ちょうどで書き出す。隣に重なる別オブジェクトは写り込まない。
- **白線対策**：書き出し前に、複製したオブジェクトを「可視範囲を整数px へ内側スナップした矩形」で**事前ラスタライズ（144ppi＝2倍相当）**してから書き出す。クリップマスク付きやリンク配置の画像で出ていた **JPEGの白い1px線／PNGの薄い縁**（スクリプトが読める境界と実際の描画範囲のずれが原因）を防ぐ。整数で作られたデータは削られず、2倍書き出しはラスタライズ解像度と1:1なので画質も落ちない。
- ドキュメントが未保存のときだけ、書き出し先を選ぶダイアログが出る。
- 実行後に **書き出した件数**を通知する。
- 対象レイヤーが無い／空のときは、その旨を知らせて何もしない。

> `.jsx` と `_shared.jsxinc` は**同じフォルダに置いたまま**にすること（`#include` で相対参照している）。
> このリポジトリのフォルダごと好きな場所に置けばよい（下記いずれの実行方法でも相対参照は解決される）。

## 実行方法

### A. すぐ試す（Presets 不要・管理者権限不要）
任意の場所（このリポジトリのまま）から都度実行できる。
- **ファイル > スクリプト > その他のスクリプト…**（`Ctrl+F12`）で `.jsx` を選んで実行。
- または `.jsx` を Illustrator のウィンドウへ**ドラッグ&ドロップ**。

この方法は「都度実行」専用。**アクションには記録できない**（下記の落とし穴を参照）。

### B. アクション＋再生ボタン（▶）で常用する ※要インストール
アクションに記録するには、スクリプトを **Presets のスクリプトフォルダにインストール**しておく必要がある（管理者権限が必要）。

1. `.jsx` 4本と `_shared.jsxinc` を、Illustrator のスクリプトフォルダへコピーする（例）:

   ```
   C:\Program Files\Adobe\Adobe Illustrator [バージョン]\Presets\ja_JP\スクリプト\
   ```

   `Program Files` 配下は管理者権限が要る。権限が無ければ IT 担当に上記コピーを依頼する。
   `_shared.jsxinc` もメニューには出ないが `#include` で必要なので一緒に入れる。
2. Illustrator を再起動 → 「ファイル > スクリプト」メニューに4本が並ぶ。
3. 「ウィンドウ > アクション」でアクションパネルを開き、新規アクション作成 → 記録開始。
4. **ファイル > スクリプト > 対象スクリプト**（メニューに並んだもの）を実行。
5. ■（記録停止）。各スクリプトごとに別アクションを作る（`move-img` / `export-img` …）。

以降は、アクションパネルで対象アクションを選んで再生ボタン（▶）を押せば実行できる。

> **落とし穴：アクション記録は「その他のスクリプト…」を記録してくれない。**
> アクションに記録できるのは Scripts メニューに名前で並ぶ＝インストール済みのスクリプトだけ。
> 「単体では動くのにアクションに入らない」場合はこれが原因。B の手順でインストールすること。
>
> インストール後にスクリプトを更新したら、スクリプトフォルダへ再コピー（上書き）が必要。
